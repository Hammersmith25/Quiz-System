# Quizify – Full-Stack Flashcard & Learning Platform 🗂️

Quizify is an interactive web application designed for creating, managing, and studying flashcards. Built with a decoupled architecture, it features a robust **Django REST Framework** backend and a dynamic **Angular** frontend.

## 🚀 Key Features

* **JWT Authentication:** Secure user login and registration using JSON Web Tokens.
* **Deck Management:** Create, update, and delete categories (decks) and individual cards.
* **Study Mode:** "Study-card" UI with randomized question sequences for effective learning.
* **API-First Design:** Fully documented endpoints, ready for testing via Postman.

## 🛠 Tech Stack

* **Backend:** Python 3.x, Django 5.x, Django REST Framework (DRF)
* **Frontend:** Angular 17+, TypeScript, SCSS
* **Database:** SQLite (Development)
* **API Testing:** Postman (VS Code Extension / Desktop)
